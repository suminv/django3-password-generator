What is a password generator? A password generator is a tool that automatically generates a password based on guidelines that you set to create strong and unpredictable passwords for each of your accounts. Good password generators do the following: Adjust guidelines to fit different sites' unique password requirements Generate strong passwords using secure technology with built-in randomness Are integrated into a password manager like Dashlane to create, manage, and easily use all of your strong passwords
